Scala.js is Scala

Scala.js is Scala

  • Can compile any valid Scala (no reflection)
  • Can use Scala libraries directly (with some small changes to the build)
  • Great IDE supports
    • Reliable refactoring
    • Correct code completion
    • Catch most errors when typing

Scala.js interpolates with JavaScript

  • Scala.js compiles to/inherits some good parts of JavaScript
    • Highly optimized JavaScript output
    • Fully asynchronous
  • Very easy to use any JavaScript libraries in Scala.js
    • React.js, Angular.js
    • D3.js
    • JQuery
    • Google Map

...or it is easy to create your own

    
trait Date extends js.Object {
  def format(pattern: String): String = js.native
}

@JSName("moment")
object Moment extends js.Object {
  def apply(): Date = js.native
}

console.log(Moment().format("MMMM Do YYYY"))

  // shared interface
  trait Api {
    def add(x: Int, y: Int, z: Int): Int
  }
  // Server
  object Server extends autowire.Server...
  object ApiImpl extends Api
    def add(x: Int, y: Int, z: Int) = x + y + z
  }
  // client-side callsite
  object Client extends autowire.Client...
  Client[Api].add(1, 2, 3).call(): Future[Int]
  //         |    |             |
  //         |    |             The T is pickled and wrapped in a Future[T]
  //         |    The arguments to that method are pickled automatically
